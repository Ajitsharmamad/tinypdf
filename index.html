Tool Descriptions — What each tool does


        
PDF Compressor

        
TinyPDF's compressor reduces PDF file size by enabling PDF object-stream compression and removing nonessential embedded objects where possible. It works fully in your browser using pdf-lib, so your files never leave your device. Use the compressor to prepare files for WhatsApp, email attachments, or to reduce storage use. For best results, try different compression targets and combine with image optimization if your PDF contains many high-resolution images.


        
PDF Merger

        
Merge multiple PDFs into a single document without uploading anything. TinyPDF merges pages client-side preserving original page order and basic metadata. Ideal for combining scanned documents, receipts, or multi-part reports into a single shareable PDF.


        
PDF Splitter

        
Split your PDF into single-page files or extract sections quickly. The splitter creates downloadable page files, perfect for sharing single pages or extracting attachments without editing software.


        
PDF → JPG

        
Convert each PDF page into high-quality JPG images using PDF.js rendering. This is useful for extracting page visuals, sharing pages as images on social platforms, or preparing images for OCR tools.


        
JPG → PDF

        
Combine multiple images into a single printable PDF. TinyPDF embeds images into pages preserving their resolution and aspect ratio so the output is suitable for printing or digital archiving.


        
Rotate PDF

        
Rotate one or all pages in a PDF (left, right or 180°) client-side. Useful for fixing scanned documents or reorienting pages before sharing.


        
PDF → Text Extractor

        
Extract readable text from PDFs using PDF.js. Works well for text-based PDFs and some scanned PDFs depending on embedded text layers. For full OCR on scanned images, use a dedicated OCR service.


        
CSV → Excel

        
Convert CSV files to downloadable .xlsx workbooks with one click. The converter parses standard CSV formats including quoted fields and returns a clean Excel file ready for editing.

About TinyPDF Tools

        
TinyPDF Tools provides free, fast, and private PDF utilities that run fully in your browser. No server upload — your files never leave your device. Designed for fast sharing over WhatsApp and quick editing tasks like compressing large PDFs to smaller sizes, merging multiple documents, and splitting pages into standalone files.


        

          
Frequently Asked Questions

          
Q: Is TinyPDF safe?

          
A: Yes — everything runs client-side. Files are not uploaded to any server.

          

          
Q: Will compressed PDF be readable?

          
A: Compression attempts to preserve visual quality using PDF object stream compression and removing nonessential objects. Results vary by file.
